# PH Node Service Template

Template para micro-serviços Node.js com TypeScript, Express e Nodemon.

## Criar novo serviço

```
npx degit nodejs-service-template novo-servico
cd novo-servico
npm install
npm run dev
```

## Estrutura

- src/index.ts
- src/health.ts
- nodemon.json
- tsconfig.json
- .env.example
